# References

## Disc Specifications

Compact Discs are defined by a collection of standards called the **Rainbow Books**. Each book in the collection defines the specification for a specific format of CD. The following is the list of standards and the CD format they define.

| Book         | Format                      | Designation (latest)              |
| ------------ | --------------------------- | --------------------------------- |
| Red Book     | CD-DA                       | IEC 60908 Ed. 2.0 B:1999          |
| Orange Book  | CD-MO, CD-R & CD-RW         |                                   |
| Yellow Book  | CD-ROM & CD-ROM XA          | INCITS/ISO/IEC 10149:1995 (S2019) |
| Green Book   | CD-i                        |                                   |
| Blue Book    | E-CD / CD+ / CD Extra       |                                   |
| Purple Book  | DDCD (DD-ROM, DD-R & DD-RW) |                                   |
| Scarlet Book | SACD                        |                                   |
| White Book   | CD-i Bridge, VCD & SVCD     | IEC 62107:2000                    |

> Note: Other "Books" did exists (e.g. the Beige Book), but these were proprietary standards never released to the public.

### Orange Book

The orange book is split up into a series of specifications that define the capabilities of various writeable discs (CD-MO, CD-R, CD-RW). Below is a list of the applicable texts.

- **Orange Book Part I**: CD-MO (CD magneto optical)
  - CD-MO standard describing 1x nominal CD speed
- **Orange Book Part II Volume 1**: CD-WO (CD write once, renamed to CD Recordable)
  - CD-R standard describing 1x, 2x and 4x nominal CD speed
- **Orange Book Part II Volume 2**: Multi-Speed Compact Disc Recordable
  - CD-R standard describing up to 48x nominal CD speed
- **Orange Book Part II Volume 3**: High Capacity Compact Disc Recordable
  - CD-R standard describing high capacity CD-R
- **Orange Book Part III Volume 1**: Compact Disc ReWritable
  - CD-RW standard describing 1x, 2x, and 4x nominal CD speed
- **Orange Book Part III Volume 2**: High-Speed Compact Disc ReWritable
  - CD-RW standard describing between 4x and 10x nominal CD speed
- **Orange Book Part III Volume 3**: Ultra-Speed Compact Disc ReWritable
  - CD-RW standard describing between both 8x and 24x nominal CD speed (Ultra-Speed) and between 8x and 32x nominal CD speed (Ultra-Speed Plus)
